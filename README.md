# Box Office Revenue Predictor

Predicts how much a movie will make at the box office using pre-release information (budget, cast, genre, release timing). XGBoost model trained on 1,919 films from TMDB.

**[→ Try the live demo](https://letsseeyourfakemovie.streamlit.app/)**

## Results

| Model | R² Score |
|-------|----------|
| Linear Regression | 0.62 |
| Random Forest | 0.62 |
| **XGBoost** | **0.71** |
